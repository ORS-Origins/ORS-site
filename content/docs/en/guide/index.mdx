## Server Philosophy

The original intention of this server is to create a harmonious, friendly, and peaceful legitimate Minecraft community with the server as the carrier. The server advocates the concept of mutual assistance and promotes polite, normal, and friendly communication. The server will also guarantee every player's gaming experience as the first principle of server management. We hope all players who join this server can clearly understand this concept and take practical actions to implement it. This server will also follow the original intention and operate stably and continuously.

## Server Rules

To ensure a good gaming experience for all players, please follow these rules:

### Basic Rules

1. **Pay Attention to Social Etiquette**: No insults, discrimination, or harassment
2. **No Cheating**: Do not use any cheat clients or mods
3. **No Malicious PVP**: Do not engage in malicious PVP behavior
4. **No Griefing**: Do not maliciously destroy other players' builds
5. **No Stealing**: Do not steal other players' items

### Building Rules

1. Do not build randomly near others' builds
2. Keep large builds at a distance from residential areas
3. Public builds require admin approval

### Violations

Players who violate the rules will face the following penalties based on severity:

- Warning
- Temporary ban
- Permanent ban

## Server Important Notes

Please read this section carefully to avoid unnecessary trouble.

1. Pay attention to social etiquette.
2. This server prohibits any form of cheating.
3. This server prohibits malicious PVP behavior.
4. This server only allows one large-scale public project at a time. Before completing the previous large-scale public project, the next one cannot be started. Large-scale public projects will be announced server-wide, and all players are obligated to participate in the construction.
