Welcome to ORS, browsing this handbook will help you further understand ORS, so that you can better integrate into this harmonious server.

## Server Philosophy

The original intention of this server is to create a harmonious, friendly, and peaceful legitimate Minecraft community with the server as the carrier. The server advocates the concept of mutual assistance and promotes polite, normal, and friendly communication. The server will also guarantee every player's gaming experience as the first principle of server management. We hope all players who join this server can clearly understand this concept and take practical actions to implement it. This server will also follow the original intention and operate stably and continuously.

## Server Notes

Please read this section carefully to avoid unnecessary trouble.

1. Please pay attention to social etiquette.
2. This server prohibits any form of cheating.
3. This server prohibits malicious PVP behavior.
4. This server only allows one large-scale public project at the same time. Before completing the previous large-scale public project, the next one cannot be started. Large-scale public projects will be announced server-wide when they start, and all players are obligated to participate in the construction of large-scale public projects.
